# Stage 1 - Stone Concept (nach Werkbank-Unlock)

## Ziel
Nach der Werkbank soll der Spieler einen klaren, mittleren Loop haben:
1. Organisches Material in `Compost` umwandeln.
2. Daraus `Dirt` und `Packed Soil` aufbauen.
3. Mit `Crude Mallet` erste Steingrundlage erzeugen.
4. `Cobblestone` und erstes Steinwerkzeug erreichen.

## Tempo-Absicht
1. Nicht schnell: Kein direkter Holz-/Stein-Skip.
2. Nicht langsam: Jede Schleife liefert spuerbaren Fortschritt.
3. Erwartung: ca. 30-60 Minuten bis stabiler Stone-Start (je nach Effizienz/RNG).

## Kernloop
1. `Leaf Threads -> Organic Dust`
2. `Organic Dust -> Compost Pile`
3. `Organic Dust + Resin -> Compost Pulp`
4. `Compost Pile -> Dirt`
5. `Dirt + Compost Pulp -> Packed Soil`
6. `Packed Soil + Crude Mallet -> Stone Grit`
7. `Stone Grit -> Cobblestone -> Furnace -> Stone Pickaxe`

## Anti-Stuck Design
1. Resin bleibt ueber `Wood Shavings + Leaf Threads` craftbar.
2. Fiber/String-Quest als Nebenzweig bei Engpaessen.
3. JEI-Info beschreibt jede Stufe inkl. Herkunft und Nutzung.
