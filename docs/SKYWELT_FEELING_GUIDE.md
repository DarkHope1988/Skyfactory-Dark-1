# Skywelt Feeling Guide (Creative Direction)

## Zielgefuehl
Spieler sollen das Gefuehl haben, aus Nichts eine lebende Himmelszivilisation zu bauen.

## Atmosphaere
1. Leere als Chance:
   - Der Void ist bewusst Teil der Identitaet, nicht nur Hintergrund.
2. Insel als Story:
   - Jede neue Plattform zeigt sichtbar, wie weit der Spieler gekommen ist.
3. Technomagischer Kontrast:
   - Mechanische Bereiche + arkan/magische Bereiche auf derselben Insel.

## Spielererfahrung pro Phase
1. Early:
   - Ueberleben, Improvisation, Materialknappheit.
2. Stone:
   - Stabilisierung, erste Routine.
3. Mid:
   - Spezialisierung (Power/Magic/Automation beginnt).
4. Late:
   - Insel wird zu Fabrik + Ritualraum.
5. End:
   - Grossprojekte, Mastery und Prestige.

## Wiedererkennbare Merkmale
1. Materialkreislaufe statt einfacher Erzsuche.
2. Vertikales Base-Design als Progressionssignal.
3. Branch-Fortschritt beeinflusst Inseloptik und Spielweise.
4. Endgame-Rezepte verbinden alle Zweige.

## Design-Check pro neuem Feature
1. Verstaerkt es das Skywelt-Gefuehl?
2. Foerdert es Entscheidungen statt nur Grind?
3. Ist es im Tier+Branch-System klar verortet?
4. Ist der Fortschritt im Spiel sichtbar kommuniziert?
