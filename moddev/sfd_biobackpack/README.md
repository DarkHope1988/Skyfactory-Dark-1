# SFD Bio Backpack (Forge 1.20.1)

Custom addon mod for Skyfactory Dark.

## Features (v0.1.0)
- New item ID: `sfd_biobackpack:bio_backpack`
- Opens on right-click in air
- Exactly 9 internal slots
- Inventory persists in item NBT

## Build
1. Open this folder in IntelliJ/Eclipse as Gradle project.
2. Ensure Java 17 is used for Gradle.
3. Generate/run Gradle wrapper (`gradlew`) from IDE or local Gradle.
4. Build with `build` task.
5. Copy generated JAR from `build/libs` to the instance `mods` folder.
